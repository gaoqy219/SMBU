中国人的虚岁

一、生词讲解

虚岁
1. 英语翻译：virtual age (Chinese traditional age calculation)
2. 俄语翻译：традиционный китайский возраст
3. 中文解释：中国人的传统年龄计算方式，一出生就算1岁，每过一个农历春节就再加1岁。
4. 例句：他周岁20岁，虚岁22岁。
周岁
1. 英语翻译：legal age (calculated from birth date)
2. 俄语翻译：юридический возраст
3. 中文解释：按照出生日期计算的年龄，每过一次生日就长1岁。
4. 例句：这个孩子今天满周岁，家人给他办了生日派对。
概念
1. 英语翻译：concept; idea
2. 俄语翻译：понятие; идея
3. 中文解释：对事物的基本认识或想法。
4. 例句：“虚岁”是中国特有的年龄概念。
计算
1. 英语翻译：calculate; compute
2. 俄语翻译：рассчитывать; считать
3. 中文解释：根据一定的方法算出数量或结果。
4. 例句：我们需要按照规则计算虚岁。
迷惑
1. 英语翻译：confused; puzzled
2. 俄语翻译：запутанный; озадаченный
3. 中文解释：心里不明白，不知道怎么回事。
4. 例句：很多人对虚岁的计算方法感到迷惑。
似是而非
1. 英语翻译：apparently true but actually false
2. 俄语翻译：похожий на правду, но ложный
3. 中文解释：看起来好像是对的，实际上是错的。
4. 例句：“虚岁就是周岁加一岁”是似是而非的说法。
农历
1. 英语翻译：lunar calendar
2. 俄语翻译：лунный календарь
3. 中文解释：中国传统的历法，根据月亮的运行制定，有春节、除夕等节日。
4. 例句：春节是农历的正月初一。
除夕
1. 英语翻译：New Year's Eve (lunar calendar)
2. 俄语翻译：последний день лунного года
3. 中文解释：农历一年的最后一天，是中国重要的传统节日。
4. 例句：他出生在除夕，一出生就算1岁，大年初一就2岁了。
大年初一
1. 英语翻译：the first day of the lunar New Year
2. 俄语翻译：первый день лунного Нового года
3. 中文解释：农历正月初一，也就是春节的当天。
4. 例句：大年初一这一天，很多人会给长辈拜年。
春节
1. 英语翻译：Spring Festival (lunar New Year)
2. 俄语翻译：Весенний фестиваль
3. 中文解释：中国最重要的传统节日，在农历正月初一，是计算虚岁的重要时间点。
4. 例句：每过一个春节，虚岁就会增加1岁。
下旬
1. 英语翻译：last ten days of a month
2. 俄语翻译：последние десять дней месяца
3. 中文解释：一个月的最后十天。
4. 例句：他的生日在农历十二月下旬，还没满月虚岁就2岁了。
满月
1. 英语翻译：one month old (baby)
2. 俄语翻译：одномесячный малыш
3. 中文解释：这里指婴儿出生满一个月。
4. 例句：这个宝宝还没满月，虚岁已经2岁了。
大寿
1. 英语翻译：grand birthday (for elderly people)
2. 俄语翻译：великая годовщина рождения
3. 中文解释：年长的人重要的生日，比如七十大寿、八十大寿。
4. 例句：爷爷今年过七十大寿，全家都很热闹。
提前
1. 英语翻译：in advance; ahead of time
2. 俄语翻译：заранее; раньше времени
3. 中文解释：比预定的时间早。
4. 例句：很多老人会提前两年过自己的大寿。
显然
1. 英语翻译：obviously; clearly
2. 俄语翻译：очевидно; явно
3. 中文解释：很容易看出来，非常清楚。
4. 例句：那种说法显然是不完全正确的。
实际
1. 英语翻译：actual; practical
2. 俄语翻译：фактический; практический
3. 中文解释：真实存在的，不是想象的。
4. 例句：在实际计算中，虚岁不只是周岁加一岁。
或许
1. 英语翻译：maybe; perhaps
2. 俄语翻译：может быть; возможно
3. 中文解释：有可能，也许。
4. 例句：或许只有中国人有两个年龄。
其实
1. 英语翻译：in fact; actually
2. 俄语翻译：на самом деле
3. 中文解释：表示真实的情况，和表面看起来不一样。
4. 例句：其实，虚岁不只是周岁加一岁。
并且
1. 英语翻译：and; besides
2. 俄语翻译：и; кроме того
3. 中文解释：表示进一步的情况，连接两个句子。
4. 例句：他出生在除夕，一出生就算1岁，并且大年初一又加1岁。
因此
1. 英语翻译：therefore; so
2. 俄语翻译：следовательно; поэтому
3. 中文解释：因为前面的情况，所以有后面的结果。
4. 例句：春节是计算虚岁的重要时间点，因此每过春节虚岁就加1岁。
理解
1. 英语翻译：understand; comprehend
2. 俄语翻译：понимать
3. 中文解释：明白别人的意思或事情的道理。
4. 例句：知道了虚岁的计算方法，我们就理解老人为什么提前过寿了。

二、课文

有人说，在全世界，或许英语：maybe; perhaps | 俄语：может быть; возможно只有中国人有两个年龄英语：age | 俄语：возраст，一个周岁英语：legal age | 俄语：юридический возраст，一个虚岁英语：virtual age | 俄语：традиционный китайский возраст。对于“周岁”是怎么回事，可能一般人还能说得清楚。而虚岁怎么“虚”，可是件很容易让人迷惑英语：confused; puzzled | 俄语：запутанный; озадаченный的事情。很多人认为，周岁加一岁得出的结果就是虚岁。这是一个似是而非英语：apparently true but actually false | 俄语：похожий на правду, но ложный的说法。

其实英语：in fact; actually | 俄语：на самом деле，很多时候，“虚岁”这个概念英语：concept; idea | 俄语：понятие; идея一般只用于男人，也就是“男进女满”，意思是，男人用虚岁计算英语：calculate; compute | 俄语：рассчитывать; считать年龄，女人用实岁计算年龄。而且，在实际英语：actual; practical | 俄语：фактический; практический的计算中，虚岁也不只是周岁加一岁那么简单。

虚岁的计算方法是这样：一个人一出生就算一岁；如果一个人正好出生在农历英语：lunar calendar | 俄语：лунный календарь年最后一天（除夕英语：New Year's Eve | 俄语：последний день лунного года），那么不但一出生就算一岁，并且英语：and; besides | 俄语：и; кроме того一到大年初一英语：the first day of lunar New Year | 俄语：первый день лунного Нового года就又要加一岁，这样算的话，这个孩子到了满实岁一岁时，他的虚岁就已经是三岁了。

因此英语：therefore; so | 俄语：следовательно; поэтому，在计算虚岁时，春节英语：Spring Festival | 俄语：Весенний фестиваль是个特别重要的时间点，每过一个春节，虚岁就应该加上一岁。如果一个人的生日是农历的12月下旬英语：last ten days of a month | 俄语：последние десять дней месяца，那他还没有满月英语：one month old | 俄语：одномесячный малыш，虚岁就到两岁了。那种认为虚岁就是周岁加一岁的看法显然英语：obviously; clearly | 俄语：очевидно; явно是不完全正确的。

知道了这个道理，我们就会理解英语：understand; comprehend | 俄语：понимать，为什么很多老人常常会提前英语：in advance | 俄语：заранее两年，过自己的七十大寿英语：grand birthday | 俄语：великая годовщина рождения、八十大寿了。
