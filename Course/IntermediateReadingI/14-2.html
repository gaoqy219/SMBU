令人担忧的汉字书写

一、生词讲解

教育部
1. 英语翻译：Ministry of Education
2. 俄语翻译：Министерство образования
3. 中文解释：中国管理教育事务的政府部门。
4. 例句：教育部发布了新的课程标准。
全日制
1. 英语翻译：full-time
2. 俄语翻译：полный рабочий день (в образовании)
3. 中文解释：整个白天都进行的学习或工作形式，比如全日制学校。
4. 例句：他在一所全日制中学读书。
课程标准
1. 英语翻译：curriculum standard
2. 俄语翻译：стандарт учебного плана
3. 中文解释：规定课程内容、要求的官方文件。
4. 例句：老师按照课程标准准备教学内容。
规范
1. 英语翻译：standard; normative
2. 俄语翻译：стандарт; нормативный
3. 中文解释：符合规定的、正确的样子。
4. 例句：我们要写规范的汉字。
端正
1. 英语翻译：upright; neat
2. 俄语翻译：прямой; аккуратный
3. 中文解释：（字或姿势）整齐、不歪斜。
4. 例句：他写的字很端正。
整洁
1. 英语翻译：neat; tidy
2. 俄语翻译：аккуратный; порядочный
3. 中文解释：干净、整齐。
4. 例句：他的作业本页面很整洁。
下降
1. 英语翻译：decline; drop
2. 俄语翻译：падение; снижение
3. 中文解释：（水平、数量等）变低。
4. 例句：最近气温在持续下降。
制定
1. 英语翻译：formulate; establish
2. 俄语翻译：разрабатывать; устанавливать
3. 中文解释：做出（规则、计划等）。
4. 例句：我们需要制定新的班级规则。
等级
1. 英语翻译：rank; level
2. 俄语翻译：ранг; уровень
3. 中文解释：按高低分的级别。
4. 例句：这个考试设置了不同的等级。
合格
1. 英语翻译：qualified; up to standard
2. 俄语翻译：квалифицированный; соответствующий стандарту
3. 中文解释：符合规定的要求。
4. 例句：他的汉字书写测试成绩合格了。
提笔忘字
1. 英语翻译：forget how to write a character when holding a pen
2. 俄语翻译：забыть, как написать знак, беря ручку
3. 中文解释：拿着笔想写字，却忘了字的写法。
4. 例句：现在很多人都有提笔忘字的问题。
错别字
1. 英语翻译：wrong or misspelled characters
2. 俄语翻译：неправильно написанные знаки
3. 中文解释：写错的字或用错的字。
4. 例句：写作文要避免写错别字。
网友
1. 英语翻译：netizen; online friend
2. 俄语翻译：сетевой пользователь; интернет-друг
3. 中文解释：在网络上交流的人。
4. 例句：他认识了很多志同道合的网友。
测试
1. 英语翻译：test; quiz
2. 俄语翻译：тест; проверка
3. 中文解释：检查知识或能力的小型考试。
4. 例句：学校对新生进行了汉字书写测试。
负责
1. 英语翻译：be responsible for; in charge of
2. 俄语翻译：быть ответственным за; руководить
3. 中文解释：承担某项工作的责任。
4. 例句：李老师负责我们班的语文教学。
形容
1. 英语翻译：describe; characterize
2. 俄语翻译：описывать; характеризовать
3. 中文解释：用语言说出事物的特点。
4. 例句：他用“吓人”来形容学生的字迹。
字迹
1. 英语翻译：handwriting; script
2. 俄语翻译：почерк; рукописный текст
3. 中文解释：字的笔画和形状样子。
4. 例句：他的字迹清晰易认。
比例
1. 英语翻译：proportion; ratio
2. 俄语翻译：пропорция; соотношение
3. 中文解释：部分和整体之间的数量关系。
4. 例句：不及格学生的比例很高。
方块
1. 英语翻译：square; block
2. 俄语翻译：квадрат; блок
3. 中文解释：四方形的形状。
4. 例句：中国汉字是方块文字。
构建系统
1. 英语翻译：construction system
2. 俄语翻译：система конструирования
3. 中文解释：指汉字由哪些笔画组成的规则。
4. 例句：汉字有自己独特的构建系统。
笔画
1. 英语翻译：stroke (of a Chinese character)
2. 俄语翻译：штрих (китайского знака)
3. 中文解释：写汉字时的一笔。
4. 例句：“人”字有两个笔画。
字形
1. 英语翻译：character form; glyph
2. 俄语翻译：форма знака; глиф
3. 中文解释：汉字的形状。
4. 例句：我们要记住每个汉字的字形。
把握
1. 英语翻译：grasp; master
2. 俄语翻译：осваивать; понимать
3. 中文解释：掌握、控制。
4. 例句：他能很好地把握汉字的结构。
结构
1. 英语翻译：structure; composition
2. 俄语翻译：структура; композиция
3. 中文解释：事物各部分的安排方式。
4. 例句：这个字的结构比较复杂。
信息时代
1. 英语翻译：information age
2. 俄语翻译：информационная эпоха
3. 中文解释：以信息为主要资源的时代。
4. 例句：我们生活在信息时代。
代替
1. 英语翻译：replace; substitute
2. 俄语翻译：заменять; подставлять
3. 中文解释：用一个事物替换另一个事物。
4. 例句：电脑慢慢代替了手写记录。

二、课文

在中国教育部英语：Ministry of Education | 俄语：Министерство образования《全日制英语：full-time | 俄语：полный рабочий день (в образовании)语文课程标准英语：curriculum standard | 俄语：стандарт учебного плана》中明确英语：clearly | 俄语：ясно规定英语：stipulate | 俄语：предписывать，书写英语：write | 俄语：писать汉字的最低要求是：规范英语：standard | 俄语：стандарт、端正英语：upright | 俄语：прямой、整洁英语：neat | 俄语：аккуратный。

但是，教育部一项调查英语：survey | 俄语：опрос显示英语：show | 俄语：показывать，在全国3000多名教师中，60%的人认为，自己学生的汉字书写水平英语：level | 俄语：уровень在下降英语：decline | 俄语：падение；这些学生（1.2万余名）中，有65%的人表示，应当赶快制定英语：formulate | 俄语：разрабатывать汉字书写等级英语：rank | 俄语：ранг标准，好让自己明确怎样写字才算合格英语：qualified | 俄语：квалифицированный。

“书写难看英语：ugly | 俄语：уродливый、提笔忘字英语：forget how to write a character when holding a pen | 俄语：забыть, как написать знак, беря ручку、写错别字英语：wrong characters | 俄语：неправильно написанные знаки”，是目前常见的三大书写问题。在互联网上，不久前，有一项调查，共有4102名网友英语：netizen | 俄语：сетевой пользователь参加英语：participate | 俄语：участвовать，结果显示：37%的人经常英语：often | 俄语：часто提笔忘字，很多不难的字都忘了怎么写；22%的人要写字时，首先英语：first | 俄语：сначала想到的是用电脑，而不是笔；16%的人觉得，除了名字写得还行，其他字基本英语：basically | 俄语：в основном难看不能看；13%的人去外面听课英语：attend a class | 俄语：посещать занятие或者开会英语：have a meeting | 俄语：вести совещание，最怕英语：most afraid | 俄语：наиболее бояться的就是记笔记。

2009年6月，武汉（Wǔhàn）大学对2100多名大一新生进行了第一次汉字书写测试英语：test | 俄语：тест，负责英语：be responsible for | 俄语：быть ответственным за测试的老师用这样的话形容英语：describe | 俄语：описывать一些学生的字迹英语：handwriting | 俄语：почерк：“吓人！”测试结果显示，有320名学生写字“不及格”，也就是说，连“规范、端正、整洁”都没有办法达到，比例英语：proportion | 俄语：пропорция很高，占参加测评全部英语：all | 俄语：все人数的15%。

中国汉字是方块英语：square | 俄语：квадрат文字，它有自己的“构建系统英语：construction system | 俄语：система конструирования”（指一个字由什么笔画英语：stroke | 俄语：штрих组成）和“笔画系统”（指笔画放在什么位置英语：position | 俄语：позиция）。这就是说，即使你了解了一个字的字形英语：character form | 俄语：форма знака，也不一定能很好地把握英语：grasp | 俄语：осваивать它的结构英语：structure | 俄语：структура。比如明天的“明”，这个字的左边是“日”，右边是“月”；但它的写法不是把这两部分放在一起就可以了，要想写得好看，“日”字要写得靠上面一点儿才会好看。

汉字书写是一门艺术英语：art | 俄语：искусство，但是在信息时代英语：information age | 俄语：информационная эпоха的今天，它正慢慢被电脑所代替英语：replace | 俄语：заменять。
